Course Outline

Module 1 :
Working with database and understanding structured query language (SQL)

  • Understanding the fundamentals of relational database
  • Data types and relational keys in SQL
  • Writing Select, Create and Drop queries
  • Types of Joins and case statements
  • Working with date functions and other syntaxes
  • Writing nested and multi-nested queries

Module 2:
Spreadsheet application using spreadsheet application for analysis

  • Connecting to data sources
  • Formulas and basic excel functions
  • Sorting, filters and advance filters
  • Cell referencing and conditional formatting
  • VLOOKUP, SUMIFs and other key functions
  • Pivot tables and charts

Module 3:
Data visualization and insights using tableau

  • Introduction to tableau and data connection in tableau
  • Working within tableau environment (Data source panel, schema panel, table or sheets)
  • Gartner structure and data types in tableau
  • Difference between tableau field and database field
  • Data cleansing in tableau, joins and cardinalities
  • Working within visualization panel
  • Calculated fields and working with date functions
  • Building visualizations, dashboard and using dashboard actions
